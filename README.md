# EditText  [![](https://jitpack.io/v/aliletter/edittext.svg)](https://jitpack.io/#aliletter/edittext) 
EditText is a control based on the official secondary development, which adds six attributes on the basis of the original, and is more flexible in use.[中文文档](https://github.com/aliletter/EditText/blob/master/README_CHINESE.md)
## Instruction
### Code Sample
```Java
    <com.aliletter.edittext.EditText
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:drawableLeft=""
        app:drawable_height=""
        app:drawable_width=""
        app:drawable_marginLeft=""
          />
```
## How to
To get a Git project into your build:
### Step 1. Add the JitPack repository to your build file
Add it in your root build.gradle at the end of repositories.   [click here for details](https://github.com/aliletter/CarouselBanner/blob/master/root_build.gradle.png)
```Java
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
### Step 2. Add the dependency
Add it in your application module build.gradle at the end of dependencies where you want to use.   [click here for details](https://github.com/aliletter/CarouselBanner/blob/master/application_build.gradle.png)
```Java
	dependencies {
            ...
	        compile 'com.github.aliletter:edittext:v1.0.0'
	}
```
